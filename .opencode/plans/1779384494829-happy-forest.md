# Setup Mistral La Plateforme sebagai Free AI Provider untuk OpenCode

## Context
- Provider sebelumnya (DeepSeek big-pickle, deepseek-v4-flash) error karena `reasoning_content` bug
- Nemotron-3-super-free bekerja tapi ingin alternatif free tier lain
- **Mistral La Plateforme dipilih** karena free tier 1B tokens/bulan - sangat generous

## Mistral Free Tier Info (TERKONFIRMASI)
- **Tokens/month**: 1 Billion (1B)
- **RPM**: 2 requests per minute
- **TPM**: ~500.000 tokens per minute
- **Model include**: Semua model termasuk Codestral (coding-specialist)
- **Credit card**: Tidak required (phone verification saja)

## Plan Setup

### Step 1: Setup Provider di OpenCode
 Jalankan `/connect` di opencode CLI:
 ```
 opencode
 /connect
 ```
 Pilih "Mistral" dan masukkan API key

### Step 2: Verifikasi
 Setelah connect, test dengan model Codestral

### Step 3: (Optional) Setup sebagai Default
 Jika perlu set default model, edit `~/.config/opencode/opencode.json`

## Files to Modify
- `/home/ubuntu/.config/opencode/opencode.json` (jika perlu adjust)

## Verification Steps
1. Run `/connect` → Mistral → masukkan API key
2. Pilih model Codestral atau Mistral Large
3. Test dengan query sederhana
4. Verify no errors in tool calls
